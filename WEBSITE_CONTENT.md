**To request a change:**
1. Find the section you want to update
2. Write your new text in the `→ Change to:` line below each item
3. Leave `→ Change to:` blank if you are happy with the current text
4. Send the completed file back and we will apply all changes

**Rules for filling in:**
- Only edit the `→ Change to:` lines — do not delete the `Current:` lines
- If a piece of text should be **removed entirely**, write `REMOVE` in the change line
- If you want to **add** something new (e.g. a new FAQ), add it at the bottom of that section with a note
- Numbers, phone numbers, and email addresses can be changed here too
